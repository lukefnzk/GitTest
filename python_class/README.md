# Python 클래스 기초 연습

아주 기초부터 단계별로 따라가는 Python `class` 학습용 샘플.
번호 순서대로 읽고, 각 파일을 직접 실행하며 출력 결과를 확인하세요.

## 학습 순서

| 파일 | 배우는 것 |
|------|-----------|
| `01_basic_class.py` | 클래스 정의, `__init__`(생성자), `self`, 속성, 메서드 |
| `02_instances_and_self.py` | 인스턴스를 여러 개 만들기, 왜 `self` 가 필요한가 |
| `03_class_vs_instance_var.py` | 클래스 변수(공유) vs 인스턴스 변수(개별) |
| `04_inheritance.py` | 상속, 메서드 오버라이드, `super()`, 다형성 |
| `05_auto_food_daily.py` | 종합 예제 — 하루 식단 관리 클래스 (AutoFoodDaily 맛보기) |
| `06_function_vs_class.py` | 함수 버전 vs 클래스 버전 비교, 그리고 둘의 혼용 |
| `07_property_and_str.py` | `__str__`(보기 좋은 출력), `@property`(속성처럼 쓰기 + 값 검사) |

## 실행 방법

```
cd python_class
python 01_basic_class.py
```

외부 라이브러리가 필요 없는 순수 Python 예제라 바로 실행됩니다.
